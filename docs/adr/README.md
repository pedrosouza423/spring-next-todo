# ADRs — Architectural Decision Records

Registros de decisões arquiteturais — "por que escolhemos X e não Y".

## Template

```markdown
# ADR NNNN — [Decision title]

**Status:** Proposed | Accepted | Deprecated | Superseded by ADR NNNN  
**Date:** YYYY-MM-DD

## Context
O que motivou esta decisão?

## Decision
O que foi decidido?

## Consequences
O que muda? Quais são os trade-offs?
```
